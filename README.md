
## Folder Stracture of Project

- **dist** - has the production build this gets generated when we run build command
- **node_modules** - stores the project dependencies
- **public** - has static files (images and icons)
- **src** - main source code folder (where application code is written) 
  - **assets** - stores static assets used in app
  - **components** - contains react components like button, input
  - **utils** - used for helper functions
  - **index.css** - global css file for styling
  - **main.tsx** - main entry point of the application

- **index.html** - main html file where react app is mounted
- **package.json** - contains project metadata, scripts and dependencies.
- **pnpm-lock.ymal**: keeps the dependencies versions consistent
 - **tsconfig.json** - for typescript configuration
- **vite.config.json** - configuration file for the vite tool
